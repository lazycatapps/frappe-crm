# frappe-crm

Starter project generated by LazyCAT hack tooling.

## Getting Started

### Prerequisites

- Make
- Git
- lzc-cli (install with `npm install -g @lazycatcloud/lzc-cli`)

### Common Commands

```bash
make help
make info
make clean
make lpk
make deploy
make uninstall
```


### Docker Commands

```bash
make docker-build
make docker-push
make docker-run
```


## Next Steps
